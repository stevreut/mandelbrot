Center

It is possible to save all of the information for generating a fractal image with this tool as a single sequence of characters in a special format known as JSON. Some of the features of this tool will eventually produce a JSON configuration for images and this configuration can then be stored and used later to reproduce the image by pasting that JSON text into the "Enter as JSON" field.

Following is a sample JSON text. Note that the elements required purposes of this site ("center", "width", and "limit") are included: 

{"center":[-1.4798237066909197,0.0005146283038554526],
 "width":2.384185791015625e-8,
 "limit":3300}

Try copying the text above, pasting it into the "Enter as JSON" box, and then pressing the "Use JSON" button.
